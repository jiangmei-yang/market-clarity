# Critical Loop 09：可探索的股票研究证据工作台

日期：2026-07-23

## 重新评分前的批判

本轮不把新增功能计为用户价值。课程评委和三类用户模拟都发现同一问题：

- 新用户默认看到文字概览，不知道真正应该先看价格、事件还是来源；
- 有研究经验的用户无法在主图逐日读取价格、成交量和同期基准；
- 评委看不到“外部说法 → 公开来源 → 事件时间 → 价格对照 → 持仓影响 → 决策审查”的连续证据；
- 原图虽有价格、基准和事件，但缺少悬停读数与尺寸控制，仍像静态课程展示。

## 已实施

- 股票研究页默认打开“价格与事件”；
- 1 月、3 月、1 年继续使用真实历史数据；
- 增加逐日十字线与悬停卡，显示日期、收盘、当日涨跌、同期沪深300和成交量；
- 支持键盘左右键逐日检查；
- 支持将图表从约 286px 放大到约 430px；
- 图上的事件标记可点击、可键盘确认，并与事件详情联动；
- 保留事件后 1 日、5 日、同期基准和持仓金额的机械对照；
- 保留“时间相邻不代表因果”的固定边界；
- 上游历史接口补充开盘、最高、最低字段，为后续 K 线视图准备真实数据，不在缺失时猜测。

## 验收

- 自动化测试：105/105 通过；
- 默认活动标签：价格与事件；
- 真实数据：贵州茅台 22 个交易日、1 个可定位正式事件；
- 键盘读数示例：2026-07-22，收盘 1305.00，当日 −0.23%，同期沪深300 −0.46%，成交量 65,181；
- 图表高度：286px → 430px；
- 页面横向溢出：0；
- 事件标记已暴露为可访问按钮。

## 诚实评分

这轮只提高可运行 MVP 与 Demo 的可理解性，不改变外部验证结论。产品控制范围内的完成度有所提升，但课程总分仍不能宣称 95：

- 外部参与者仍为 0；
- 真实模型固定题集仍未形成可发布结果；
- 价格候补或付款行为证据仍为 0。

下一轮应优先完成真实模型评测运行轨迹和首批外部用户，而不是继续扩展新的金融模块。
