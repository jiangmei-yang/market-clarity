# Critical Loop 11 执行提示词：Agent 必须完成任务

你正在审查 Market Clarity（安心看股）的 Goal-to-Workflow Agent。不要用功能数量、计划面板或模型连接状态代替任务完成。

依次模拟三类真实用户：

1. 新手：“每周只有半小时，不想真实交易，也不知道先看什么，帮我安排工作台。”
2. 持仓用户：“读取我的持仓，找出最大单一仓位和行业风险；只告诉我核对什么，不改工作台。”
3. 研究用户：“研究 600519 最近价格、公告和财务变化，列出需要核对的事实，不改页面，不给买卖建议。”

逐项检查：

- 是否只选择完成目标所需的最少工具；
- 是否擅自创建提醒、观察列表、规则或模拟组合；
- 是否把不存在的工具写成已执行；
- 是否真正调用行情、财报、公告和持仓工具；
- 是否显示来源、数据时间、缺失字段和降级状态；
- 是否把缺失行业错误计算为行业集中；
- 是否提供用户可以立即核对的结果，而不是“请前往其他页面”；
- 是否保留人工确认、无自动交易和无收益承诺边界；
- 是否出现横向溢出、重复空卡或大段模型文字。

每发现一个问题，先修业务契约，再修文案和视觉。修改后用相同原句复测。不得把模拟角色结果写成外部用户证据。
