# 用户测试计划

## 研究目标

验证安心看股是否让普通 A 股投资者更快、更准确地理解一只股票的主要风险，并促使其在交易前停下来核实或记录理由。

## 参与者（5 人）

1. 用户父亲：中老年、有 A 股经验、目标核心用户。
2. 另一位 55 岁以上、常用东方财富或同花顺的投资者。
3. 35–54 岁、有持仓但不熟悉技术指标的投资者。
4. 近两年刚开始投资的新手。
5. 熟练使用雪球/行情软件的进阶用户，作为反向压力测试。

避免只访谈亲友中支持项目的人。至少 2 位参与者应由非团队成员介绍。

## 访谈流程（每人 35 分钟）

### 1. 背景（5 分钟）

- 最近一次买卖股票前做了什么？
- 最难判断或最担心什么？
- 平时在哪看行情、公告和观点？
- 最近一次因为信息太多或遗漏而后悔是什么？

不介绍产品，不问“你会不会需要”。先收集真实过去行为。

### 2. 无引导任务（15 分钟）

让用户边操作边说出想法，主持人不教按钮位置。

1. “请告诉我这个产品是做什么的。”
2. “请查看贵州茅台最近有什么值得注意。”
3. “请找出风险提示为什么出现。”
4. “如果你想以后再看，怎么做？”
5. “请记录一个你自己的判断，或解释为什么不愿记录。”

### 3. 对照任务（5 分钟）

用参与者平时的工具完成同一问题：“贵州茅台现在最值得注意的风险是什么？”记录时间和答案，不要求产品间功能一致。

### 4. 回顾（10 分钟）

- 哪一处最有帮助？请举具体例子。
- 哪一处不可信或让你困惑？
- 哪些信息可以删除？
- 下次准备买卖时，你会在哪个时刻打开它？
- 如果不能用，你会回到什么方法？
- 是否愿意未来一周真实使用 3 次？让用户自己选择，不用口头赞美代替承诺。

## 记录指标

| 指标 | 记录方式 | 成功门槛（5 人探索样本） |
|---|---|---:|
| 价值理解 | 10 秒后复述 | ≥4/5 提到“整理重点/风险” |
| 搜索完成 | 无帮助完成 | ≥4/5 |
| 首次价值时间 | 打开到读完总结 | 中位数 ≤60 秒 |
| 风险理解 | 说出风险+证据 | ≥4/5 |
| 误解率 | 是否把风险当预测/建议 | ≤1/5 |
| 自选发现 | 无帮助找到 | ≥3/5 |
| 记录意愿 | 实际完成而非口头表示 | ≥2/5 |
| SUS-lite | 3 题 1–5 分 | 易用性均值 ≥4 |
| 一周复访 | 后续真实打开 ≥2 天 | ≥2/5 |

## 观察模板

- 参与者编号、年龄段、投资年限、常用工具。
- 首句话、首次点击、停顿超过 5 秒的位置。
- 主持人提供了几次帮助。
- 对“风险等级”“演示数据”“一分钟看懂”的原话解释。
- 想删除的信息和期待增加的信息。
- 最后一次真实投资决策能否映射到产品场景。

## 研究纪律

- 不在任务前解释页面。
- 不把“很好”“会用”当证据；优先记录实际完成、时间和复访。
- 不因一个用户要求立刻开发；同一障碍出现 ≥3 人，才进入 P0/P1。
- 用户要求荐股、预测或自动交易时，记录需求但不突破安全边界。
- 获得同意后才录屏；不记录证券账号、身份证、银行卡或真实密码。
