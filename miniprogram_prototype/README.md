# 安心看股微信小程序原型

> **状态：已冻结 / 实验性代码。** 当前 MVP 的唯一正式交付平台是 Streamlit 网页。本目录不再继续扩展、调试或用于正式发布；恢复条件见项目根目录 `PAUSED_PLATFORM_WORK.md`。

这是课程展示用的独立前端原型，不是已经备案或公开上架的小程序。它只覆盖：欢迎、自然语言规则设置、规则确认、五项交易输入、理由确认、风险卡和修改前后对比。

默认 `demoMode: true`，可用微信开发者工具以测试 AppID 打开本目录。演示数据只用于交互展示。

需要连接后端时：

1. 将 FastAPI 部署到已备案、支持 HTTPS 的域名。
2. 在微信小程序后台把域名加入服务器域名白名单。
3. 在 `app.js` 只填写公开的 HTTPS API 基址，并把 `demoMode` 设为 `false`。
4. OpenAI Key、Tushare Token、数据库密钥和小程序 AppSecret 必须只存在于服务器端。

本原型不包含券商连接、下单、完整行情、复杂图表、自选股或真实微信登录。
