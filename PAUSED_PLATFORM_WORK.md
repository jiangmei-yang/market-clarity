# 已暂停的平台工作

**状态日期：2026-07-21。网页版是当前唯一正式交付。**

## 已冻结资产

### 微信小程序体验原型

位置：`miniprogram_prototype/`

已包含欢迎、规则描述、规则确认、交易计划、理由确认和审查卡的页面骨架，以及面向后端 API 的适配文件。它是实验性代码，未完成真实联调、备案、隐私审核、正式发布或多设备验收；当前不再扩展和调试，也不影响 Streamlit 启动。

未来可复用：核心路径信息架构、字段名称、确认式交互和 API 数据形状。

### PWA 探索

现有 `static/manifest.webmanifest`、图标占位和离线说明页只保留为探索资产。Streamlit 的路由、Service Worker 和持续连接限制使其不适合作为本阶段的完整 PWA；当前只承诺手机友好的云端网页。

### Android / iOS

本项目没有建立完整 Android、Android WebView、iOS、Flutter 或 React Native 应用。`ANDROID_READINESS.md` 仅是未来路线分析，不代表正在交付移动 App。

## 明确暂停

- 小程序前端开发、真机调试、备案和上架；
- Android/iOS/WebView 壳；
- Flutter、React Native、多端同步；
- 完整 PWA 改造、应用商店配置和多平台 UI 适配；
- 仅为未来平台服务的基础设施重构。

## 恢复条件

满足以下条件后才重新决定入口，不自动恢复：

1. 完成至少 10–20 位目标用户的网页版测试；
2. 核心审查流程稳定，并确认用户愿意重复使用；
3. 识别出用户最常使用的三项任务；
4. 有证据表明网页入口造成了显著流失，或课程明确要求特定平台；
5. 重新评估备案、审核、隐私、维护时间和预算。

届时仍优先复用 `src/decision_review/`、数据模型、确定性计算和后端 API，不复用 Streamlit 页面代码。
