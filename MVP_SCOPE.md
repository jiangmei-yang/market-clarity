# 安心看股 MVP 范围（冻结版）

## 唯一正式交付

当前只交付一个产品：**面向 A 股普通投资者的网页版交易前 AI 决策审查工具**。

它不推荐股票、不预测涨跌、不连接券商、不执行交易，也不进行心理诊断。核心价值不是“告诉用户买什么”，而是帮助用户在操作前把理由说清楚、核实信息缺口、计算金额影响，并对照自己预先确认的提醒边界再做决定。

## 本轮必须形成的闭环（P0）

1. 用自然语言描述个人提醒边界；
2. 系统转成结构化规则，用户逐项确认或修改；
3. 输入股票、操作、金额、理由和失效条件五项；
4. 系统把理由拆成事实、推断、未核实信息和情绪表达；
5. 用户确认系统理解；
6. 普通代码计算仓位、集中度和 10%/20%/30% 下跌情景；
7. 有限 RAG 返回可追溯资料及覆盖限制；
8. 生成不含买卖建议的决策审查卡；
9. 用户选择维持、修改或稍后再看；
10. 保存前后变化，并可导出匿名用户测试 CSV。

核心流程在没有 OpenAI API Key、实时行情或外网时仍可用固定演示数据完成。

## P1（只在 P0 稳定后）

- 扩充有限、可信、可维护的 RAG 资料；
- 完善资料日期和引用；
- 历史审查记录与简单行为报告；
- 手机浏览器细节适配；
- 云部署、访问密码和数据备份说明。

## 暂不做（P2）

- 完整行情、自然语言选股和量化回测；
- 自动交易、券商连接和全网新闻抓取；
- 小程序正式发布、Android、iOS、Flutter、React Native；
- 完整 PWA、多端同步、多用户商业账户和支付；
- 复杂心理状态诊断或大规模生产架构。

## 架构边界

允许把规则解析、交易计划模型、规则计算、理由审查、有限检索、报告和研究记录从 Streamlit 页面分离；不做微服务、Kubernetes、Redis、多数据库抽象或跨端 UI 框架。

## 成功标准

- 新用户 2 分钟内完成个人规则设置；
- 60 秒内完成一次交易计划审查；
- 能纠正系统对原话的理解；
- 能看到金额风险和个人规则冲突；
- 能自行修改或延迟，并保留前后变化；
- 90 秒固定 Demo 无 Token、无实时接口也能完成；
- 电脑和手机浏览器均能操作；
- 能导出不含直接身份信息的测试数据。
